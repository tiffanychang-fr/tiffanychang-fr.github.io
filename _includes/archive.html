• Fancy Adventure

    Fancy Adventure is a mini browser game built with javaScript and P5.js library. You will discover four levels which represents four seasons with missions to complete.

  • Coffee In A Box

    Coffee In A Box is an online shop which sells monthly coffee bean subscription box made with mongoDB, express.js, node.js, and styling with Bootstrap.

  • Piggy Back

    Piggy Back is an online community which connects foreigners and locals to help settling down process easier. A marketplace made with react.js, express.js, node.js, mongoDB, and styling with Bootstrap by Titos.

  • Catch Disney

    Catch Disney is an browser game perfect for a little break. Select a character to start, catch them as many as you want. It is built with javascript, css and html.

  • Simon Game

    Simon Game is a browser game to challenge your memory, originally invented by Ralph H. There are four colors, press the color in the correct order, repeat the patterns to get higher level. It is built with javascript, css and html.

  • Tindog

    Tindog is a mock project to find your dog a date. It is built with bootstrap, css and html.
